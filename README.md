# Student Management API

A REST API built using Flask and SQLite to perform CRUD operations on student data.

## Technologies Used
- Python
- Flask
- SQLite

## API Endpoints

GET /students  
Returns all students

POST /students  
Adds a new student

PUT /students/<id>  
Updates student details

DELETE /students/<id>  
Deletes a student

## How to Run the Project

1. Install dependencies
pip install flask

2. Run the application
python app.py

3. Open in browser
http://127.0.0.1:5000/students
